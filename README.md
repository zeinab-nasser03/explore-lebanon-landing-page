# Explore Lebanon Landing Page

A modern and responsive travel landing page showcasing some of Lebanon's most beautiful destinations.

## Features

* Responsive Design
* CSS Flexbox Layout
* CSS Grid Gallery
* Glassmorphism Navigation Bar
* Interactive Hover Effects
* Mobile-Friendly Layout
* Beautiful Destination Cards

## Destinations Featured

* Beirut
* Jeita Grotto
* Byblos
* Batroun
* South Lebanon
* Cedars of God

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome

## Learning Objectives

This project was developed as part of a Web Development Internship and demonstrates:

* Flexbox for layout and alignment
* CSS Grid for content organization
* Responsive Web Design
* Modern UI/UX principles
